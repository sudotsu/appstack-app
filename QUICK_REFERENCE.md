## üö® Common Rejection Reasons (AVOID THESE)

1. **Missing privacy policy** - You need one (app uses Claude API)
2. **Misleading screenshots** - Show actual app, not mockups
3. **Broken functionality** - Test thoroughly before submitting
4. **Wrong content rating** - Be honest in questionnaire
5. **Copyright issues** - Don't use others' logos/brands
6. **Incomplete store listing** - Fill out ALL required fields
