- **Smart Hints**: Progressive hint system that guides without spoiling

  - Tailwind v4 Dynamic Styling
  - State Management
  - Form Handling & Validation
  - Next.js Routing Concepts
  - Async Data Fetching
  - Custom Hooks & Composition

## Install as Android PWA

Once deployed, you can install this as a Progressive Web App on Android:

1. Open the deployed URL in Chrome on your Android device
2. Tap the menu (three dots) in the top right
3. Select "Install app" or "Add to Home Screen"
4. The app will be added to your home screen and function like a native app
5. It will work offline and save your progress locally
